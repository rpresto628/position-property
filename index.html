Static positions are per hmtl flow. These elements are considered to be positioned.
Relative position is the same as static unless you give extra values.
Giving values to top, right, bottom & left will move the elements. Theses valies are relative to its default position.
Behaves like relative except that it is relative to its parent element. Top, right, bottom & left rive it a new position.
Fixed position will move to a new position and stay there even if content around it is scrolled through.
